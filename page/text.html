文本

链接

作用在.link类上。初始的<a>标签已经消除本身自带的颜色表现，而链接会还原本来的颜色。

    
<a href="javascript:;" class="link">链接</a>

段落

在容器加上.p类，里面的<p>标签则会有相应表现。

消失的舊時光一九四三在回憶的路上時間變好慢

老街坊小弄堂是屬於那年代白牆黑瓦的淡淡的憂傷

消失的舊時光一九四三回頭看的片段有一些風霜
老唱盤舊皮箱裝滿了明信片的鐵盒裡藏著一片玫瑰花瓣

    
<div class="p">
	<p>消失的舊時光一九四三在回憶的路上時間變好慢</p>
	<p>老街坊小弄堂是屬於那年代白牆黑瓦的淡淡的憂傷</p>
	<p>消失的舊時光一九四三回頭看的片段有一些風霜<br>老唱盤舊皮箱裝滿了明信片的鐵盒裡藏著一片玫瑰花瓣</p>
</div>

标签

作用在.tag类上，可以与颜色类搭配。

默认 蓝色 
    
<span class="tag">默认</span>
<a class="tag blue-o">蓝色</a>

徽章

作用在.badge类上，当内容为空时不会显示。

好友9 私信20+ 黑名单

	<a class="btn blue-o">好友<span class="badge">9</span></a>
	<a class="btn yellow-o">私信<span class="badge black-o">20+</span></a>
	<a class="btn black-o">黑名单<span class="badge"></span></a>
</div>
