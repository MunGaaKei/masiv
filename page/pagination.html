分页

生成

简单的构建，只需要一个容器。

    
M('#page-a').page(1, 7);

参数设置

page()函数可以通过第三个参数进行其它配置

    
M('#page-b').page(1, 13, {
	prev: '<i class="iconfont icon-left"></i>',
	next: '<i class="iconfont icon-right"></i>',
	click: function( p ){
		this.classList.add('is-loading');
		setTimeout(function(){
			M('#page-b').page(p, 13);
		}, 500);
	}
});

只显示上一页和下一页。

    
M('#page-c').page(1, 19, {
	siblings: false,
	prev: '上一页',
	next: '下一页'
});

参数说明

page(p, pages[, options])函数的参数说明。

第一个参数p为当前页码数；

第二个参数pages为总页码数；

第三个参数[options]为一个对象，用于配置分页的样式。

参数默认值说明
hreffalse传入url字符串会在页码上添加href属性并且值为该值加上页码组成的字符串。
prevfalse传入字符串显示上一页。
nextfalse传入字符串显示下一页。
siblings2当前页的相邻页码数，当值为false时则只显示上页和下页
pageClass"page"页码按钮的基础类名
activeClass"active"当前页码按钮的激活类名
dots省略页码显示字符串
clickfalse页码点击回调函数
