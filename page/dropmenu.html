下拉菜单

使用

触发器使用[masiv-dropmenu]属性标识。

你不喜欢的那些东西
  • 运动
  • 植物
  • 动物
    
<div masiv-dropmenu>
	<a class="btn">你不喜欢的那些东西<span class="caret"></span></a>
	<ul class="dropmenu">
		<li>
			<label class="label-inline group" style="width: 220px;">
				<input type="text" class="input" autocomplete="off">
				<a class="btn blue">寻找</a>
			</label>
		</li>
		<li class="li">运动</li>
		<li class="li">植物</li>
		<li class="li">动物</li>
	</ul>
</div>

鼠标悬停触发

使用[masiv-dropdown]属性替代[masiv-dropmenu]属性即可，并且可以通过类.leftward、.rightward、.upward来控制弹出方向。

电影
  • 爱情片
  • 恐怖片
    • 午夜凶铃
    • 咒怨
    • 安娜贝尔
    
<div masiv-dropdown>
	<a class="btn blue">电影</a>
	<ul class="dropmenu upward">
		<li class="li">爱情片</li>
		<li>
			<div class="col-x" masiv-dropdown>
				<a class="li navs">恐怖片<i class="caret"></i></a>
				<ul class="dropmenu rightward">
					<li class="li">午夜凶铃</li>
					<li class="li">咒怨</li>
					<li class="li">安娜贝尔</li>
				</ul>
			</div>
		</li>
	</ul>
</div>
