JS 
    
M('#masiv-slide').slide({
    display: 3,
    scroll: 2,
    prev: '<i class="iconfont icon-left"></i>',
    next: '<i class="iconfont icon-left"></i>',
    navs: true,
    responsive: {
        '720': { display: 2, scroll: 1 },
        '360': { display: 1 }
    },
    afterSlide: function( current ){
        document.getElementById('current').textContent = '当前索引值：'+ current;
    }
});

参数属性说明

参数默认值说明
prev'<'后退按钮，值为false时不显示
next'>'前进按钮，值为false时不显示
navsfalse导航，值为true时显示
navsTrigger'click'导航按钮触发事件类型
looptrue轮播是否循环
display1每屏显示的个数
scroll1每次切换的个数
start0初始化后的开始位置
autofalse是否自动轮播
duration3000单位ms，自动轮播的时间间隔
pauseOnHoverfalse自动轮播时是否鼠标悬停时暂停轮播
speed300单位ms，切换的过渡时间
easing'ease'切换的过渡动画效果
responsiveundefined接受一个对象参数，对象键名为breakpoint值，值则为设置
beforeSlideundefined接受一个函数作为轮播切换前触发的回调函数
afterSlideundefined接受一个函数作为轮播切换后触发的回调函数

方法

每次实例化后，可以继续使用slide(fnName[, param])方法来操作对应的轮播实例。

或者通过调用slide('slide')方法，该方法返回一个对象，该对象有更多对于该轮播实例的操作方法与属性。

1
2
3
next
JS 
    
var $function = M('#masiv-function').slide({
    prev: false,
    next: false
});
var $slide = $function.slide('slide');

$function.slide('next');
$slide.next();

$function.slide('goto', 0);
$slide.goto(0);

prev()

切换至上一个

next()

切换至下一个

goto( i )

i int：根据索引切换到指定位置

destroy()

注销该轮播实例

current

返回该轮播的当前项索引值

length

返回该轮播的数量
