树

引入JS

    
<script src="js/M.tree.js"></script>

另外，由于该插件默认使用iconfont字体图标，所以还需要引入相应的CSS。或者你可以使用别的字体图标，在这个情况下你需要重新设置部分属性（参考设置说明）。

初始化

使用一个<ul>作为容器，传入数据生成对应节点。

    HTML 
        
<ul id="menu"></ul>
    JS 
        
var data = [{
        name: '介绍',
        attr: { type: '属性' },
        open: true,
        children: [{
            name: '起步',
            href: 'start.html'
        }]
    }, ..., {
        icon: '<i class="iconfont icon-plugin"></i>',
        name: '插件',
        children: [{
            icon: '<i class="iconfont icon-tree"></i>',
            name: '树',
            href: 'tree.html'
        }, ...]
    }];

M('#menu').tree({
    data: data
});

    初始参数属性说明

    参数默认值说明
    data[]必需，节点数据结构
    checkboxfalse开启选择框功能
    checked'<i class="iconfont icon-checked"></i>'选中按钮图标
    unchecked'<i class="iconfont icon-unchecked"></i>'未选中按钮图标
    folder'<i class="iconfont icon-folder"></i>'文件夹图标
    item'<i class="iconfont icon-file"></i>'子项图标
    toggler'<i class="iconfont icon-down"></i>'展开收缩图标
    targetundefined同<a>标签属性

    节点属性设置说明

    属性默认值说明
    name''必需，节点名称
    attrundefined对象 {} 类型节点自定义属性
    checkedfalse是否默认选中该节点
    hrefundefined链接
    targetundefined同<a>标签属性，会覆盖初始化时的设置
    iconundefined字符串类型，节点图标，设置了会覆盖初始化时的设置
    childrennull数组 [] 类型，同初始化数据格式，子节点数据
    openfalse是否默认打开该目录
