特点

轻量：

核心文件 masiv.css压缩后仅~24kb。

简洁：

框架注重于基础样式重置，剔除了不必要的样式语句。意味着基础样式的设计极简与拥有更强的扩展性。

语义化：

推崇HTML5语义化标签，类名也通过语义化来命名。

响应式：

在不同屏幕分辨率下会有不同的布局显示。

功能：

提供丰富且极可能需要的组件功能。

移动端：

除了CSS，JS在移动端也做了事件处理，如tap、swipe、press。

文件引入

下载文件后，在页面里引入masiv.css文件，部分功能则需要引入masiv.js进行支持。


以下是一个简单的MASIV页面模板。

    
<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <meta name="viewport" content="initial-scale=1,maximum-scale=1,minimum-scale=1">
    <title>MASIV</title>

    <link rel="stylesheet" href="css/masiv.css">
</head>
<body>

    <!-- HTML -->

    <script src="js/masiv.js"></script>
</body>
</html>
