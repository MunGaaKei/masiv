输入框

引入JS，下载。

    
<script src="js/plugins/input.min.js"></script>

另外，由于该插件默认使用iconfont字体图标，所以还需要引入相应的CSS。

富文本编辑器

在输入框添加属性[masiv-input]，值为editor。

HTML 

<div masiv-input="editor"></div>

文件上传

在输入框添加属性[masiv-input]，值为file，其中[data-title]指定按钮文本。

HTML 
    
<div class="field">
    <label class="label-inline">
        <span class="label">附件：</span>
        <input type="file" class="input round" data-title="上传附件" masiv-input="file">
    </label>
</div>

日期控件

在输入框添加属性[masiv-input]，值为calendar。

HTML 
    
<div class="field">
    <label class="label-inline">
        <span class="label">入学日期：</span>
        <input type="text" class="input round" masiv-input="calendar">
    </label>
</div>
