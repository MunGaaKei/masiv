播放器

引入JS

    
<script src="js/M.player.js"></script>

另外，由于该插件默认使用iconfont字体图标，所以还需要引入相应的CSS。或者你可以使用别的字体图标，在这个情况下你需要重新设置部分属性（参考设置说明）。

初始化

自动生成可以指定一个容器加上[masiv-player]属性，并指定[data-src]作为播放地址。
另外，还可以通过[data-loop]等属性来进行设置。

HTML 
    
<div id="masiv-demo" masiv-player data-src="resource/video.mp4"></div>
JS 
    
M('#masiv-video').player();

更多可选参数的初始

不使用[masiv-player]属性，用JS命令生成可以进行更多选项的设置。

HTML 
    
<div id="#masiv-params"></div>
JS 
    
M('#masiv-params').player({
    play: '<b>PLAY</b>',
    pause: '<b>PAUSE</b>',
    poster: '../resource/poster.jpg',
    src: '../resource/video.mp4',
    title: '示例视频'
});

参数属性说明

参数默认值说明
title
[data-src]		''视频描述，默认位于视频左上角
src
[data-src]		必需播放地址
control
[data-control]		true是否显示控制栏，值为false时隐藏
loop
[data-loop]		false是否循环播放
autoplay
[data-autoplay]		false是否自动播放
timetrue是否显示播放时间，值为false时隐藏
play'<i class="iconfont icon-play"></i>'播放按钮
pause'<i class="iconfont icon-pause"></i>'暂停按钮
stop'<i class="iconfont icon-stop"></i>'停止按钮
volume'<i class="iconfont icon-volume"></i>'音量按钮
mute'<i class="iconfont icon-mute"></i>'静音按钮
fullscreen'<i class="iconfont icon-fullscreen"></i>'全屏按钮
fullscreen-exit'<i class="iconfont icon-fullscreen-exit"></i>'退出全屏按钮
forward'<i class="iconfont icon-forward"></i>'快进按钮
backward'<i class="iconfont icon-backward"></i>'后退按钮
color'blue-o'进度条颜色类
vol.5初始音量大小，范围为[0, 1]
forwardUnit3快进/后退单位，单位秒s
tiptrue是否显示提示，值为false时隐藏
