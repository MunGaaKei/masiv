主题

选择

在/themes文件夹里，存放着主题预设方案，修改里面的scss并编译即可。下面是/default文件夹说明：

    
├── default
|   ├─ var.scss             /* 变量设置 */
|   ├─ reset.scss           /* 默认样式重置 */
|   ├─ components.scss      /* 组件样式 */
|   ├─ plugin.scss          /* 插件样式 */
|   ├─ custom.scss          /* 自定义添加的样式 */
|   └─ masiv.css            /* 默认编译出来的 css 文件，可忽略 */

命名空间

默认使用并且推荐的是简洁的无前缀类名，如：.table、.blue等。

在必要的情况下（为了某些功能必须要引入其它的框架，产生类名冲突问题），可以在var.scss文件里设置$m变量值，开启类名前缀功能。

假设$m变量值为: msv-，此时.table组件使用.msv-table类名来修饰。

部分功能组件如果需要使用到masiv.js文件，则还需要在页面的<body>标签设置属性[masiv-namespace]，值即为前缀字符串。

    
<!DOCTYPE html>
<html>
<head>
</head>
<body masiv-namespace="msv-">

    <!-- HTML Codes -->

</body>
</html>
