按钮

使用

    
<button class="btn">基础的按钮样式</button>

表现

按钮加上相应的颜色类，如.black：

可选择的颜色有：黑色、白色、蓝色、红色、黄色、粉色、绿色、灰色。

黑色 蓝色 绿色 
    
<a class="btn black">黑色</a>
<a class="btn blue">蓝色</a>
<a class="btn green plain">绿色</a>

填充的则加上.black-o类：

黑色 蓝色 黄色 红色 绿色 粉色 白色 灰色 
    
<a class="btn black-o">黑色</a>

加载状态下的按钮

加载中 每三秒钟点击我一次 
    
<a class="btn is-loading">加载中</a>
<a class="btn white-o">每三秒钟点击我一次</a>

禁用状态下的按钮

    
<button class="btn" disabled>你不可以点击我</button>

组合

多个按钮可以放入.group容器里。

蓝色 黑色 黄色 红色 
    
<div class="group round">
    <a class="btn blue-o">蓝色</a>
    <a class="btn black-o">黑色</a>
    <a class="btn yellow-o">黄色</a>
    <a class="btn red-o">红色</a>
</div>

.collapsed类可以使按钮边框合并，建议作用在子元素为边框类型的组。带有.iconfont的按钮建议在文本内容加上标签。

HOME FRIENDS SETTING
  • PROFILE
  • HELP
  • EXIT
    
<div class="group collapsed">
    <a class="btn black">
        <span>HOME</span>
        <i class="iconfont icon-home"></i>
    </a>
    <a class="btn black">
        <span>FRIENDS</span>
        <i class="iconfont icon-friends"></i>
    </a>
    <a class="btn black">
        <span>SETTING</span>
        <i class="iconfont icon-setting"></i>
    </a>
    <div masiv-dropmenu>
        <a class="btn black-o"><i class="iconfont icon-more-v"></i></a>
        <ul class="dropmenu black-o">
            <li class="li">PROFILE</li>
            <li class="li">HELP</li>
            <li class="li">EXIT</li>
        </ul>
    </div>
</div>
