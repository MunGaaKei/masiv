按钮

使用

    
<button class="btn">基础的按钮样式</button>

表现

按钮加上相应的颜色类，如.black：

可选择的颜色有：黑色、白色、蓝色、红色、黄色、绿色。

黑色 蓝色 绿色 红色 白色 黄色 
    
<a class="btn black">黑色</a>
<a class="btn blue">蓝色</a>
<a class="btn green plain">绿色</a>
<a class="btn red">红色</a>
<a class="btn white">白色</a>
<a class="btn yellow plain">黄色</a>

反显的幽灵按钮则加上.reverse类：

黑色 蓝色 黄色 红色 绿色 白色 
    
<a class="btn black reverse">黑色</a>

加载状态下的按钮

加载中 每三秒钟点击我一次 
    
<a class="btn is-loading">加载中</a>
<a class="btn blue">每三秒钟点击我一次</a>

禁用状态下的按钮

    
<button class="btn" disabled>你不可以点击我</button>

组合

多个按钮可以放入.group容器里。

蓝色 黑色 黄色 红色 
    
<div class="group round">
    <a class="btn blue">蓝色</a>
    <a class="btn black">黑色</a>
    <a class="btn yellow">黄色</a>
    <a class="btn red">红色</a>
</div>
