滚动监听

引入JS，下载。

    
<script src="js/plugins/scrollspy.min.js"></script>

使用

该插件规定导航需使用[href]属性指定节点对应关系。

  • HTML
  • JS
  • Neptune
  • Jupiter
  • Saturn

HTML

    
<ul id="scrollspy-navs sticky col-sm-2">
    <li><a href="#html" class="li">HTML</a></li>
    <li><a href="#js" class="li">JS</a></li>
    <li><a href="#neptune" class="li">Neptune</a></li>
    <li><a href="#jupiter" class="li">Jupiter</a></li>
    <li><a href="#saturn" class="li">Saturn</a></li>
</ul>

<div id="html">
    <h3>HTML</h3>
    ...
</div>

<div id="js">
    <h3>JS</h3>
    ...
</div>

参数属性说明

属性默认值说明
navs'a'导航项选择器
active'active'导航高亮类名
offset'0px'偏移值
viewportundefined目标区域节点
enterundefined接受一个函数作为目标激活时触发的回调函数
