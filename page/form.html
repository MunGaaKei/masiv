表单

每个表单行使用.field容器。

输入框

为了直接绑定标签与输入内容，输入框与文本域建议使用<label>标签包裹，并且在该标签上加上.label-input类或者.label-inline类。

.group类用于合并按钮、输入框等的圆角。

    
<div class="field">
	<label class="label-input">
		<span class="label">用户名：</span>
		<input type="text" class="input" autocomplete="off">
	</label>
	<label class="label-input">
		<span class="label">密码：</span>
		<input type="password" class="input error" autocomplete="off">
	</label>
</div>
<div class="field">
	<label class="label-inline">
		<span class="label">邮箱：</span>
		<div class="group">
			<input type="email" class="input" autocomplete="off">
			<a class="btn blue-o">检测</a>
		</div>
	</label>
</div>
<div class="field">
	<label class="label-inline">
		<span class="label">备注：</span>
		<textarea class="textarea" rows="3"></textarea>
	</label>
</div>
<div class="cols">
	<button class="btn black-o right" type="submit">提交表单</button>
</div>

选择器

[selected]代表选中项，[disabled]则代表不可选中项。如果想要使整个选择器禁用，则需在[masiv-dropmenu]容器上加上[disabled]属性。

    
<div class="field">
	<label class="label-input">
		<span class="label">星座：</span>
		<div class="select" masiv-dropmenu>
			<input type="text" class="input" readonly value="处女座">
			<span class="caret"></span>
			<ul class="dropmenu">
				<li class="li" data-value="摩羯座">摩羯座</li>
				<li class="li" data-value="处女座" selected>处女座</li>
				<li class="li" data-value="水瓶座" disabled>水瓶座</li>
				...
				<li class="li" data-value="巨蟹座">巨蟹座</li>
			</ul>
		</div>
	</label>
	<label class="label-input">
		<span class="label">生肖：</span>
		<div class="select" masiv-dropmenu>
			<input type="text" class="input">
			<ul class="dropmenu">
				<li class="li" data-value="🐒">🐒</li>
				<li class="li" data-value="🐕">🐕</li>
				<li class="li" data-value="🐖">🐖</li>
			</ul>
		</div>
	</label>
</div>

单选框

<label>使用.label-radio类。

    
<div class="field">
	<label class="label">性别：</label>
	<label class="label-radio"><input type="radio" class="radio" name="sex">男</label>
	<label class="label-radio"><input type="radio" class="radio" name="sex">女</label>
</div>
<div class="field">
	<label class="label">水果：</label>
	<div>
		<label class="label-radio"><input type="radio" class="radio" name="fruit" checked>西瓜</label>
		<label class="label-radio"><input type="radio" class="radio" name="fruit">苹果</label>
		<label class="label-radio"><input type="radio" class="radio" name="fruit">香蕉</label>
	</div>
</div>

多选框

<label>使用.label-checkbox类。

    
<div class="field">
	<label class="label-checkbox"><input type="checkbox" class="checkbox" checked>请记住我</label>
	<label class="label-checkbox"><input type="checkbox" class="checkbox">我已阅读同意条款</label>
</div>
